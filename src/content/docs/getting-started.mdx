## Creating a new project

<Tabs>
	<TabItem label="GitHub">
    <ol>
        <li>Click use template button below</li>
        <a href="https://github.com/new?template_name=sizzle_starter&template_owner=hawkkiller"
        target="_blank">
        <Image src={useTemplateButton} alt="Use Template, GitHub" height="100" width="150"/>
        </a>
        <li>Give your project a name and click `Create repository`</li>
    </ol>
    </TabItem>
	<TabItem label="Others">
    <ol>
        <li>Click `Import repository` in your service</li>
        <li>Give it a name and create a repository</li>
    </ol>
    </TabItem>
	<TabItem label="Git">
    <ol>
        <li>Clone the repo with `git clone https://github.com/hawkkiller/sizzle_starter.git`</li>
        <li>Create a repository in your provider and copy the url</li>
        <li>Change the remote url with `git remote set-url origin url`</li>
    </ol>
    </TabItem>
</Tabs>

## Setting up the project

<ol>
    <li>Clone the repo with `git clone`</li>
    <li>Open it in your IDE and run `flutter pub get`</li>
    <li>Click `Find and Replace All` in your IDE and replace `sizzle_starter`
    with your project name</li>
    <li>Run `flutter create --org yourorg.com .`</li>
</ol>

### Splash screen

<ol>
    <li>Replace `assets/images/splash.png` with your splash screen</li>
    <li>There is a new API for splash screen on Android 12+, learn about it [here](https://developer.android.com/develop/ui/views/launch/splash-screen)</li>
    <li>Run `dart run flutter_native_splash:create`</li>
</ol>

### Launcher Icons

<ol>
    <li>Replace `assets/images/icon-1024x1024` with your icon.
    You can also check the configuration in pubspec.yaml to provide different
    icons for platforms.</li>
    <li>Run `dart run flutter_launcher_icons`</li>
</ol>
